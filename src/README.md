# USAGE

## Install Poetry

```sh
$> pip install poetry
```

## Setup Poetry

```sh
$> poetry install     # install packages for project
$> poetry shell

$(flask-server-py3.11) PS D:\Your\Directory\flask-server\src>
```

## Install Packages in Poetry

```sh
$> poetry add <Your-Package-Name>
```

## Starting Server

```sh
$>
```

## Deactivate

```sh
$> deactivate
```
